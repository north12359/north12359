TERMUX搭建HEXO以及后续部署(中)

发表于2023-07-01|更新于2023-07-01|教程
|字数总计:1.7k|阅读时长:6分钟|阅读量:

前言

参考了butterfly官方教程
主题配置文件指 根目录/themes/butterfly/_config.yml
根目录配置文件指 根目录/_config.yml
让我们开始吧

安装butterfly

  1. 安装butterfly

git clone -b master https://github.com/jerryc127/hexo-themr-butterfly.git themes/butterfly

  1. 安装pug和stylus渲染

npm install hexo-renderer-pug hexo-renderer-stylus --save

  1. 修改根目录下的 _config.yml 开启主题

(```)
#Extensions

##Plugins: https://hexo.io/xxxxx

##themes: https://hexo.io/xxxx

**theme: butterfly **
(```)

设置博客个人资料

(```)
#Site

title: #标题
subtitle: #副标题
description: #个性签名
author: #作者
language: #语言
timezone: #时区
(```)

主题支持三种语言

  • dedault(EN)
  • zh-CN(简体中文)
  • zh-TW(繁体中文)

时区设置为 Asia/Shanghai 否则后续部署会报错

导航菜单

() menu: 主页: / || fas fa-home 博文 || fa fa-graduation-cap: 分类: /categories/ || fa fa-archive 标签: /tags/ || fa fa-tags 归档: /archives/ || fa fa-folder-open 生活 || fas fa-list: 分享: /shuoshuo/ || fa fa-comments-o 相册: /photos/ || fa fa-camera-retro 音乐: /music/ || fa fa-music 影视: /movies/ || fas fa-video 友链: /links/ || fa fa-link 留言板: /comment/ || fa fa-paper-plane #留言板: /messageboard/ || fa fa-paper-plane 关于笔者: /about/ || fas fa-heart ()

代码块设置

在主题文件中的_config.yml中配置一下功能

() highlight_theme: mac # darker / pale night / light / ocean / mac / mac light / false 代码主题 highlight_copy: true # 复制按钮 highlight_lang: false # 是否显示代码语言 highlight_shrink: false # true: 代码块默认不展开 / false: 代码块默认展开 | none: 展开并隐藏折叠按钮 highlight_height_limit: false # unit: px code_word_wrap: true #代码自动换行，关闭滚动条 ()

同时将站点根目录中的_config.yml的highlight设置为false

() highlight: enable: false line_number: false auto_detect: false ()

添加本地搜索

  1. 安装插件

npm install hexo-genetator-search --save

  1. 打开主题配置文件_config.yml

() #Local search local_search: enable: true labels: input_placeholder: Search for Posts hits_empty: "We didn't find any results for the search: ${query}" # 如果没有查到内容相关内容显示 ()

新建分类页面和标签页面

  1. 分类

hexo new page categories
创建后使用 vim source/categories/index.md进入菜单页
在date下面添加type=categories

  1. 标签

hexo new page tags
创建后使用 vim source/tags/index.md进入标签页
在date下面添加type=tags

自定义副标题

() #the subtitle on homepage (主頁subtitle) subtitle: enable: true #Typewriter Effect (开启打字效果) effect: true #loop (循環打字) loop: true #source調用第三方服務 #source: false 關閉調用 #source: 1 調用搏天api的隨機語錄（簡體） #source: 2 調用一言網的一句話（簡體） #source: 3 調用一句網（簡體） #source: 4 調用今日詩詞（簡體） #subtitle 會先顯示 source , 再顯示 sub 的內容 source: false #如果有英文逗号' , ',请使用转义字元 &#44; #如果有英文双引号' " ',请使用转义字元 &quot; #开头不允許转义字元，如需要，请把整個句子用双引号包住 #如果关闭打字效果，subtitle只会现示sub的第一行文字 sub: - 你在抱怨什么呢 / - 为明天到来的事，说人生像是没有意义 /这几处可以 - 没有选择会是唯一的路 /写自己喜欢的 - 这不是你自己的问题，人终归要好好去生活 /句子 ()

图片设置

图片可以用云链接或本地文件 修改主题配置文件 _config.yml

  1. 网站图标

() #Favicon（网站图） favicon: /img/favicon.png ()

  1. 头像

() avatar: img: /img/avatar.jpg #图片路径 effect: false #头像会一直转圈 ()

  1. 主页封面图片

() #The banner image of home page index_img: /img/background.jpg ()

  1. 文章顶部图

在没有设置top_img和cover会显示该图

() #If the banner of page not setting, it will show the top_img default_top_img: /img/default_top_img.jpg ()

  1. 归档页顶部图

() #归档子标签页图片 #The banner image of archive page archive_img: /img/archive.jpg ()

  1. tag页顶部图

() #tag页（标签页） tag_img: /img/tag_img.jpg ()

  1. category页顶部图

() #category页 category_img: /img/category_img.jpg ()

  1. 统一文章封面
    (```)
    cover:
    index_enable: true # 是否展示文章封面
    aside_enable: true
    archives_enable: true
    position: both # 封面展示的位置 left/right/both

#当没有设置cover时，默认展示的文章封面
default_cover:
#当配置多张图片时，会随机选择一张作为 cover. 此时写法为
/- https:
/- http:
/- http:
/- http:
/- http:
/- http:
(```)

OS:正常没有 / 我为了不让makedown误转译添加的

单独设置文章封面 在文章md文件添加

(```)

title: Hello World # 标题
tags: [hello] # 标签
categories: # 分类
description: hello word~ # 描述
top_img: /img/hello-1.png # 顶部背景图
cover: /img/hello-1.png # 文章封面

(```)

  1. 错误页面

配置后会替换无法显示的图片

() #Replace Broken Images (替換無法顯示的圖片) error_img: flink: /img/friend_404.gif post_page: /img/404.jpg ()

图片懒加载

添加插件

npm install hexo-lazyload-image --save

在根目录_config.yml添加一下内容

() lazyload: enable: true loadingImg: /img/loading.gif ()

图片大图查看

修改主题配置文件 _config.yml

() medium_zoom: false fancybox: true ()

版权样式

修改主题配置文件 _config.yml

() #copy settings #copyright: Add the copyright information after copied content (複製的內容後面加上版權信息) copy: enable: true # 是否开启网站复制权限 copyright: # 复制的内容后面加上版权信息 enable: false # 是否开启复制版权信息添加 limit_count: 50 # 字数限制，当复制文字大于这个字数限制时 ()

相关文章

在文章下面添加推送 修改主题配置文件

() #Related Articles related_post: enable: true limit: 6 # Number of posts displayed date_type: created # or created or updated 文章日期顯示創建日或者更新日 ()

打赏

给文章结尾添加打赏 修改主题配置文件

() #Sponsor/reward reward: enable: true QR_code: - img: #此处改为你的微信收款码链接 link: text: 微信 - img: #此处改为你的支付宝收款码的图片链接 link: text: 支付宝 ()

侧边栏样式

  1. 边栏位置

() aside: enable: true hide: false button: true mobile: true # display on mobile position: right # left or right ()

公告栏设置

修改主题配置文件

() card_announcement: enable: true content: This is my Blog #修改公告栏信息 ()

TOC目录

() #toc (目錄) toc: post: true page: false number: false expand: true # 是否展开 style_simple: false # for post scroll_percent: true ()

字数统计

安装统计插件
npm install hexo-wordcount --save or yarn add hexo-wordcount

修改主题配置文件

() #wordcount (字數統計) wordcount: enable: true post_wordcount: true min2read: true total_wordcount: true ()

文章分享功能

addthis sharejs addtoany任选一个开启

(```)
#Share System (分享功能)
#————————————–

#AddThis
#https://www.addthis.com
addThis:
enable: false
pubid: #访问 AddThis 官网, 找到你的 pub-id

# Share.js
# https://github.com/overtrue/share.js

sharejs:
enable: true
sites: facebook,twitter,wechat,weibo,qq #想要显示的内容

#AddToAny
# https://www.addtoany.com/
addtoany:
enable: false
item: facebook,twitter,wechat,sina_weibo,facebook_messenger,email,copy_link
(```)
